# getdxinfo

This is command line app to query DirectX feature_level for win7.

We assume the default DirectX 11 in available.


## example output

```txt

DX11 runtime found
DX11 device created
Feature level: 11_1
All supported feature levels:
  11_1
  11_0
  10_1
  10_0
  9_3
  9_2
  9_1
Adapter: NVIDIA GeForce RTX 3080 Laptop GPU
Video memory: 16177 MB

```
